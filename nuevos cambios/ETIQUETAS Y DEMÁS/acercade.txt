display	flex 
Convierte el elemento .acercade en un contenedor flexible (Flex container), lo que permite organizar sus elementos hijos (el texto y la imagen).



align-items center 
Alinea los ítems flexibles a lo largo del eje secundario (vertical). Esto asegura que la imagen y el bloque de texto se mantengan centrados verticalmente uno frente al otro.



justify-content	space-between 
Distribuye el espacio entre los ítems a lo largo del eje principal (horizontal). Coloca el primer ítem al inicio, el último al final, y reparte el espacio restante uniformemente entre ellos.



padding	80px 40px 
Añade un espacio interior de 80 píxeles arriba y abajo, y 40 píxeles a los lados.



gap 40px 
Establece un espacio de 40 píxeles entre los elementos de la sección (entre el texto y la imagen), evitando que se toquen.



flex-wrap wrap 
Permite que los ítems salten a una nueva línea (uno encima del otro) si no hay suficiente espacio horizontal. Esto es clave para que el diseño se vea bien en pantallas móviles.





.acercade .texto {
  flex: 1;
  min-width: 300px;
}


flex 1 Esta es una propiedad abreviada de Flexbox. Al establecerla en 1, le indica al navegador que este elemento (.texto) debe crecer (flex-grow: 1) y encogerse (flex-shrink: 1) para ocupar todo el espacio disponible que no esté siendo utilizado por los otros elementos (la imagen). Es decir, el bloque de texto ocupará la mayor parte del espacio horizontal.



min-width 300px	
Establece el ancho mínimo que el bloque de texto debe tener.
¿Qué hace? Asegura que, incluso cuando el texto se está encogiendo en pantallas pequeñas, nunca será más estrecho de 300 píxeles. Si el ancho total de la sección .acercade es menor que la suma de este min-width y el ancho de la imagen, el flex-wrap: wrap; que está en el contenedor padre hará que los elementos salten a una nueva línea.








.acercade .imagen {
  flex: 0 0 300px;
  text-align: center;
}




flex 0 0 300px 
Esta es la propiedad abreviada de Flexbox más importante aquí, que combina tres valores en este orden: flex-grow, flex-shrink, y flex-basis.



flex-grow: 0 Indica que el contenedor de la imagen no debe crecer para ocupar espacio adicional. Mantiene su tamaño.


flex-shrink: 0 Indica que el contenedor de la imagen no debe encogerse por debajo de su tamaño base. Mantiene su tamaño incluso si hay poco espacio.

	
flex-basis: 300px Establece el tamaño inicial del elemento en 300 píxeles (su ancho).	

El contenedor de la imagen siempre tendrá un ancho fijo de 300 píxeles.

La imagen no cambiará de tamaño si el ancho de la pantalla varía (no crecerá ni se encogerá).

La imagen estará centrada dentro de esos 300 píxeles.
