.galeria-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 25px;
  max-width: 1200px;
  margin: 0 auto;
}




display	grid 
Convierte el elemento en un contenedor de cuadrícula (Grid Container), activando todas las propiedades de CSS Grid.


grid-template-columns repeat(auto-fit, minmax(300px, 1fr))

Define el patrón de columnas:

repeat	
Repite la definición de columna.
La función repeat() le dice al navegador que repita un patrón de columna o fila. En lugar de escribir el tamaño de columna varias veces (ej: 1fr 1fr 1fr), puedes usar repeat(3, 1fr).
Al usar autofit con repeat le decimos al navegador que calcule cuántas columnas de 300px caben en el contenedor padre y luego las ajuste para que ocupen todo el largo restante si lo hay, si la pantalla es chica el mínimo será de 300px como nosotros pusimos

auto-fit 
Le indica al navegador que cree tantas columnas como quepan en el ancho disponible.


minmax(300px, 1fr) 
Define el ancho de cada columna: la columna debe ser, como mínimo, de 300px, pero si hay espacio adicional, debe ocupar 1fr (una fracción) del espacio restante. Esto significa que las columnas se estirarán para llenar el ancho total si hay espacio, pero nunca serán más estrechas de 300px.

Esto es por si la pantalla es grande las columnas se muestran a lo largo y si es chica, a no menos de 300px




.galeria-item {

  overflow: hidden;
  border-radius: 15px;
  box-shadow: 0 5px 20px rgba(0, 0, 0, 0.2);
  transition: transform 0.3s ease;
}



¿Qué es overflow: hidden;?
Es una propiedad de CSS que controla lo que sucede si el contenido de un elemento es demasiado grande para caber en su caja contenedora (es decir, si el contenido se "desborda").

Al establecerla en hidden (escondido/oculto), le indicas al navegador que:

Recorte cualquier contenido que se extienda más allá de los bordes del elemento contenedor.
