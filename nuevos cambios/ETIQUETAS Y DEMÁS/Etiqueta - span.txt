La etiqueta <span> en HTML sirve principalmente para aplicar estilos o manipular una pequeña porción de contenido dentro de un elemento en línea (como texto). 


Es un contenedor genérico en línea que no tiene ningún significado semántico especial por sí mismo. Su utilidad se manifiesta cuando necesitas segmentar contenido para, por ejemplo, darle formato específico con CSS o modificarlo con JavaScript, sin alterar la estructura del documento.



La etiqueta <span> es a menudo usada junto con los atributos class o id para:

Aplicar Estilos Específicos (CSS): Es el uso más común. Permite seleccionar una palabra, una frase o una parte de texto para aplicarle un estilo diferente al resto del párrafo o línea.

Ejemplo:
<p>Esta es una frase con una <span style="color: blue; font-weight: bold;">palabra importante</span> resaltada.</p>
